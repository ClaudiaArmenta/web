Dashboard descriptivo

Esta interfaz interactiva va dirigida para el área de soporte técnico. Su propósito es facilitar la visualización de datos relevantes obtenidos durante todo el proceso de resolución de un ticket, de principio a fin. Cabe mencionar que engloba factores tales como el tiempo de resolución, la empresa, el nivel designado, residencia, entre otros; lo cual permite tener una perspectiva más amplia del estado actual, y de esta forma poder tomar decisiones de una manera más efectiva y siempre enfocada en la mejora continua.
